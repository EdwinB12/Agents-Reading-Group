# Trust Architectures & Security for AI Agents

**Proposed by:** Community
**Date added:** 2026-03-24
**Suggested format:** paper / discussion

## Why interesting

Securing AI agents in production requires understanding trust models, prompt injection defences, and governance frameworks like NIST AI RMF. This session would cover Zero Trust principles and hands-on lab exercises for building defensible agent systems.
